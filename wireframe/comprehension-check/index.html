Comprehension Check
Screen builder note
A small pause button dropped at any point in a lesson. Clicking it opens one of six micro-check formats. Interrupts the content flow just enough to surface understanding before the learner moves on. Six formats shown as variant tiles — author chooses which format to deploy.

How it works
Placement
Author drops the button anywhere in a lesson. It sits inline — not at the end, not in a separate activity. The check happens inside the flow.
Format selection
Author picks one format per check point from the six variants. Each format surfaces understanding differently. Mix across a lesson for variety.
Gating
Nothing is gated on a check result. Learner always continues. Feedback nudges, not blocks.
Facilitator view
Aggregated responses visible in facilitator dashboard (confidence ratings, format 5 muddiest choices, format 1 predictions logged).
